# Synchronic Test Project

Цей проект є тестовим завданням, реалізованим за допомогою Webpack, Sass та JS.

## Вимоги

Перед початком роботи переконайтеся, що у вас встановлено:
- [Node.js](https://nodejs.org/) (версія 14 або вище)
- npm (зазвичай встановлюється разом з Node.js)

## Встановлення

1. Клонуйте репозиторій (якщо ви це ще не зробили).
2. Перейдіть у папку проекту:
   ```bash
   cd synchronic-test
   ```
3. Встановіть залежності:
   ```bash
   npm install
   ```

## Запуск проекту

### Режим розробки

Для запуску локального сервера розробки з гарячим перезавантаженням (HMR):

```bash
npm start
```

Сервер буде доступний за адресою [http://localhost:3000](http://localhost:3000).

### Збірка проекту

Для створення оптимізованої збірки (production build):

```bash
npm run build:prod
```

Файли збірки будуть розміщені у папці `build/`.

Для створення збірки в режимі розробки (development build):

```bash
npm run build:dev
```

## Структура проекту

Проект має наступну структуру (FSD - Feature Sliced Design спрощений):

- `src/app` - Глобальні налаштування, стилі та ініціалізація додатку.
- `src/pages` - Сторінки додатку (наприклад, Home).
- `src/widgets` - Самостійні блоки інтерфейсу (Header, GameLists тощо).
- `src/features` - Реалізація бізнес-фіч.
- `src/shared` - Перевикористовувані компоненти, утиліти, конфіги.
- `public` - Статичні файли (index.html).

## Технології

- **Webpack 5** - Збирач модулів.
- **Babel** - Транспіляція JS.
- **Sass (SCSS)** - CSS препроцесор.
- **CSS Modules** - Локалізація стилів компонентів.
